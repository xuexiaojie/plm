# 计算平台实施任务清单

## 阶段一：基础平台

1. 建立项目、名目、结构树、参数、步骤、结果核心表
2. 实现项目与名目管理 API
3. 实现树节点增删改查、移动、复制、排序 API
4. 实现全局参数与项目参数管理 API
5. 实现 Python 执行器和单节点执行能力
6. 实现整树执行与结果展示

## 阶段二：双语言执行

1. 为 `CalcStep` 增加语言和入口点字段
2. 实现 `ExecutorRegistry`
3. 实现 C# 执行器
4. 统一执行上下文和结果 JSON 协议
5. 增加执行任务表和后台 worker

## 阶段三：审批流

1. 实现审批请求与审批日志模型
2. 实现提审、通过、拒绝、撤回 API
3. 增加结果与树版本审批状态控制
4. 前端增加审批页面和状态展示

## 阶段四：对比分析

1. 增加 `step_type` 分类能力
2. 实现对比组与对比项 API
3. 实现指标抽取与报表查询 API
4. 增加图表展示和导出功能

## 阶段五：AI 分析

1. 实现 AI 分析请求与结果模型
2. 接入共享文件和现场反馈上下文
3. 实现分析请求创建和结果查询 API
4. 增加 AI 分析页面与结果展示
