# 梁式步进炉二级离线模型任务书

## 1. 任务名称

梁式步进炉二级离线模型程序开发

## 2. 任务目标

开发一套面向步进梁式加热炉的二级离线模型程序，用于根据钢坯参数、炉段参数和工艺目标，离线计算各加热区段的炉温设定值，并预测钢坯出炉温度与断面温差。

## 3. 主要参考

主参考采用博士论文《步进梁加热炉炉温综合优化控制策略研究》，并结合步进梁加热炉常见二级模型结构实现程序代码：

- 钢坯温度预报模型
- 炉温优化设定模型

## 4. 任务内容

1. 建立钢坯厚度方向一维非稳态导热模型。
2. 建立预热段、加热段、均热段分区计算模型。
3. 根据目标出炉温度生成理想温升曲线。
4. 构建离线优化目标函数。
5. 开发程序完成仿真与优化计算。
6. 输出各段炉温设定值、出炉钢温、断面温差和分段结果。

## 5. 输入数据

程序输入包括：

- 钢坯宽度、厚度、长度
- 钢坯密度、比热、导热系数、黑度
- 入炉温度、目标出炉温度
- 最大表里温差限制
- 最大升温速率限制
- 步距、步进周期
- 各区段长度、初始炉温设定值、换热系数

## 6. 输出结果

程序输出包括：

- 各段优化炉温设定值
- 各段出口表面温度、心部温度、平均温度
- 出炉表面温度、心部温度、平均温度
- 出炉温度偏差
- 表里温差
- 最大升温速率
- 能耗代理项、氧化烧损代理项、综合目标函数值

## 7. 技术路线

1. 采用厚度方向离散网格建立钢坯温度场。
2. 对每个炉段按驻留时间做显式差分迭代。
3. 在边界引入对流与辐射换热。
4. 使用温度相关等效物性修正导热过程。
5. 构建包含质量指标和能耗指标的综合目标函数。
6. 采用两阶段坐标搜索完成离线设定值优化。
