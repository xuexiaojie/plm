# 工业炉计算平台 V1.0 操作说明书

## 1. 文档目的

本文用于指导用户完成工业炉计算平台 V1.0 的日常使用，包括系统登录、项目管理、计算管理、报告下载、项目资料上传、AI 问答、审批报告、数字孪生和权限分配等操作。

本文重点说明计算类功能的界面结构、操作步骤、输入输出内容、图示结果和 PDF 计算报告样本，便于工程人员按项目完成工业炉热工、机械、水梁、烧嘴、管道、二级离线模型等计算工作。

## 2. 适用对象

本说明书适用于以下角色：

1. 项目经理：维护工程项目，查看项目资料和计算结果。
2. 工艺工程师：录入炉型参数，执行热工、传热、排烟、换热器、二级模型等计算。
3. 设备工程师：执行水梁、炉辊、步进框架、液压系统、装出钢机、装出料辊道等设备计算。
4. 审批人员：查看审批报告和计算报告，确认计算结论。
5. 系统管理员：维护角色、权限和用户授权。

通过浏览器打开系统预览地址或部署地址后，进入登录页面。

1. 在登录页输入用户名和密码。
2. 点击登录按钮进入系统主界面。

## 5. 项目管理操作

### 5.1 功能目的

`项目管理` 用于维护工程项目台账。计算管理、项目资料和 AI 问答均依赖工程项目作为业务上下文。

### 5.2 基本流程

1. 点击左侧 `项目管理`。
2. 查看已有工程项目列表。
3. 维护项目名称、项目编码、企业、项目经理、项目介绍等基础信息。
4. 保存后，项目会出现在 `计算管理` 和 `项目资料` 的工程项目下拉框中。

### 5.3 项目信息建议

建议项目介绍中包含炉型、产线、产能、主要工艺条件、改造范围和特殊设计边界，便于后续模糊查询和报告引用。

## 6. 计算管理操作

### 6.1 功能目的

`计算管理` 是平台的核心计算入口。用户先选择工程项目，再选择炉型，最后点击固定计算模块进入弹出式计算界面。

### 6.2 页面结构

`计算管理` 页面由以下区域组成：

1. `工程项目模糊查询`：输入关键词筛选工程项目。
2. `工程项目`：选择当前计算所属项目。
3. `项目信息卡片`：显示项目经理和项目介绍。
4. `炉型标签`：固定包含 `步进炉`、`辊底炉`、`环形炉`。
5. `计算模块按钮区`：显示当前炉型下的计算模块。
6. `计算弹窗`：点击计算模块后打开，用于录入参数、执行计算和下载报告。

### 6.3 工程项目选择

1. 点击左侧 `计算管理`。
2. 在 `工程项目模糊查询` 中输入项目经理、工程项目名称、企业、项目介绍或项目编码。
3. 在 `工程项目` 下拉框中选择目标项目。
4. 查看项目经理和项目介绍是否正确。
5. 项目选中后，炉型标签和计算模块按钮自动启用。

未选择工程项目前，炉型标签和计算模块按钮处于置灰状态，无法点击。

### 6.4 炉型与计算模块

#### 6.4.1 步进炉

步进炉计算模块按业务组显示：

| 组别 | 计算模块 |
| --- | --- |
| 总体热工 | 步进炉二级离线模型、加热曲线、传热计算、多工况计算 |
| 热工单体 | 排烟计算、蓄热计算、空气管道、换热器、煤气管道 |
| 水梁计算 | 水梁计算、水冷梁水冷计算、气化冷却计算、水梁垫块黑印计算 |
| 烧嘴计算 | 平焰烧嘴计算、高速烧嘴计算、蓄热烧嘴计算 |
| 炉底机械 | 步进框架计算、步进液压系统计算、装出钢机计算、装出料辊道计算 |

#### 6.4.2 辊底炉

辊底炉包含：

1. `辊底炉二级离线模型`
2. `辊强度计算`

#### 6.4.3 环形炉

环形炉包含：

1. `环形炉二级离线模型`

### 6.5 通用计算流程

1. 在 `计算管理` 中选择工程项目。
2. 点击炉型标签，例如 `步进炉`。
3. 点击具体计算模块，例如 `加热曲线`。
4. 系统打开弹出式计算界面。
5. 按界面字段录入或调整输入参数。
6. 点击计算按钮。
7. 查看结果卡片、判定状态、图示输出和工程建议。
8. 点击 `下载 PDF 计算报告` 或 `保存` 生成图文报告。

### 7.2 二级离线模型界面

适用模块：

1. `步进炉二级离线模型`
2. `辊底炉二级离线模型`
3. `环形炉二级离线模型`

#### 7.2.1 界面目的

二级离线模型用于模拟钢坯在炉内的升温过程，输出出炉表面温度、中心温度、平均温度、表心温差、升温曲线和厚度方向温度云图。

#### 7.2.2 主要输入

| 类别 | 字段示例 |
| --- | --- |
| 钢坯参数 | 厚度、宽度、长度、密度、比热、导热系数 |
| 工艺目标 | 装炉温度、目标出炉温度、总在炉时间 |
| 炉温制度 | 预热段、加热段、均热段或三段炉温 |
| 炉长分配 | 各段长度或各区长度 |
| 换热参数 | 发射率、气氛修正、接触换热修正 |
| 模型模式 | 炉温优化、仅仿真 |

#### 7.2.3 主要输出

1. 出炉表面温度。
2. 出炉中心温度。
3. 出炉平均温度。
4. 出炉温度偏差。
5. 表心温差。
6. 最大升温速率。
7. 推荐炉温设定。
8. 升温曲线。
9. 温度云图。
10. 最近 5 次计算比较。
11. 最终计算选择。

#### 7.2.4 操作步骤

1. 选择工程项目。
2. 选择炉型标签。
3. 点击对应二级离线模型。
4. 选择 `炉温优化` 或 `仅仿真` 模式。
5. 录入钢坯规格、炉温制度、在炉时间和物性参数。
6. 点击计算按钮。
7. 查看升温曲线和温度云图。
8. 点击 `查看计算对比` 查看最近 5 次计算。
9. 在对比表中选择合适结果作为最终计算。
10. 点击 `下载 PDF 计算报告` 生成报告。

### 7.3 步进炉加热曲线界面

#### 7.3.1 界面目的

`加热曲线` 用于计算钢坯在步进炉内的升温过程、热平衡、能耗、排烟温度和煤气量估算。

#### 7.3.2 参数区域

| 区域 | 主要字段 |
| --- | --- |
| 炉子参数 | 炉子产量、加热时间、出炉目标温度、各段炉温 |
| 料坯及空煤气 | 钢坯规格、钢种成分、空气参数、煤气组成、煤气热值 |
| 散热损失 | 炉墙散热、水冷散热、炉门孔洞散热、其他热损失 |

#### 7.3.3 输出内容

1. 出钢表面温度。
2. 出钢平均温度。
3. 出钢中心温度。
4. 出钢平均温差。
5. 最大温度应力。
6. 单位能耗。
7. 排烟温度。
8. 煤气量估算。
9. 加热曲线。
10. 温度云图。
11. 热平衡图。

### 7.4 步进炉排烟计算界面

#### 7.4.1 界面目的

`排烟计算` 用于计算烟道阻力、预热器损失、烟囱抽力、排烟机前压力损失、烟囱高度和出口内径。

#### 7.4.2 参数区域

1. 烟道阻力计算输入参数：竖烟道、预热器前后烟道、烟囱等分段参数。
2. 预热器与冷风修正：预热器阻力、温降、冷风吸入系数等。
3. 烟囱高度计算输入参数：出口流速、出口温度、大气压、烟囱材料等。

#### 7.4.3 输出内容

1. 各段烟道压力损失。
2. 排烟机前压力总损失。
3. 算得烟囱高度。
4. 烟囱出口内径。
5. 烟囱实际抽力。
6. 烟囱阻力损失。
7. 烟囱开度。
8. 接冷风量。
9. 图上标注式排烟系统示意图。

### 7.5 步进炉换热器界面

#### 7.5.1 界面目的

`换热器` 用于管式换热器热工核算，输出介质预热温度、烟气出口温度、阻力损失、流速和管束温度分布。

#### 7.5.2 参数区域

1. 基本参数：预热介质流量、烟气流量、入口温度、管束尺寸和烟道尺寸。
2. 核校输入参数：横向根数、纵向根数、风箱尺寸。
3. 设计输入参数：介质流速上下限、烟气流速上下限、最低预热温度。
4. 选项：计算模式、排列方式、型式、介质、行程和插入件。

#### 7.5.3 输出内容

1. 钢管平均温度。
2. 介质平均温度。
3. 介质预热温度。
4. 介质阻力损失。
5. 烟气出口温度。
6. 烟气阻力损失。
7. 温度分布图。
8. 管束结构示意图。

### 7.6 步进炉传热计算界面

#### 7.6.1 界面目的

`传热计算` 用于炉墙、炉顶、炉底、水梁包扎、管道绝热和地下砌砖烟道的多层传热核算。

#### 7.6.2 页签说明

| 页签 | 主要用途 |
| --- | --- |
| 炉墙 | 炉墙多层耐材传热与炉壳温度计算 |
| 炉顶 | 炉顶锚固结构传热和热桥修正 |
| 炉底 | 架空炉底或实心炉底传热计算 |
| 水梁包扎 | 水冷管包扎结构温度、吸热和流量计算 |
| 管道绝热 | 管道多层绝热传热计算 |
| 地下砌砖烟道 | 拱顶地下烟道散热、温降和基础温度计算 |

#### 7.6.3 输出内容

1. 炉壳温度或外壁温度。
2. 热流或散热量。
3. 炉衬厚度。
4. 炉衬造价。
5. 各层耐材热面温度。
6. 温度分布图。
7. 结构示意图。

### 7.7 步进炉水梁与专用设备计算界面

适用模块：

1. `水梁计算`
2. `水冷梁水冷计算`
3. `气化冷却计算`
4. `水梁垫块黑印计算`
5. `步进框架计算`
6. `步进液压系统计算`
7. `装出钢机计算`
8. `装出料辊道计算`

#### 7.7.1 输出特点

这类模块通常输出承载、弯矩、应力、挠度、冷却水量、产汽量、黑印温降、驱动功率、液压流量或节拍校核等结果，并配套结构或载荷示意图。

#### 7.7.2 操作要点

1. 先确认设备结构边界，例如跨距、承载辊数、行程、缸径、管径等。
2. 再录入载荷、速度、材料强度、安全系数和效率参数。
3. 计算后重点查看判定状态和安全裕度。
4. 需要归档时下载 PDF 计算报告。

### 7.8 烧嘴计算界面

适用模块：

1. `平焰烧嘴计算`
2. `高速烧嘴计算`
3. `蓄热烧嘴计算`

#### 7.8.1 主要输入

1. 总供热负荷。
2. 单只或单对烧嘴能力。
3. 燃气低位热值。
4. 空气系数。
5. 理论空气量。
6. 火焰直径、喷口速度或换向周期。
7. 余热回收效率和烟气温度。

#### 7.8.2 主要输出

1. 烧嘴数量或烧嘴对数。
2. 燃气流量。
3. 助燃空气量。
4. 火焰覆盖率。
5. 喷口直径。
6. 喷流动量。
7. 节约燃气量。
8. 空气预热温度。

### 7.9 管道与蓄热计算界面

适用模块：

1. `空气管道`
2. `煤气管道`
3. `蓄热计算`

#### 7.9.1 空气管道和煤气管道

管道计算主要输入流量、温度、压力、密度、内径、长度、摩擦阻力系数、局部阻力系数和允许压损。输出工况流量、管内流速、推荐内径、动压、沿程阻力、局部阻力和总压力损失。

#### 7.9.2 蓄热计算

蓄热计算主要输入蓄热体体积、密度、比热、烟气温度、目标排烟温度、空气入口温度、空气流量、换向周期和孔道流速。输出蓄热体质量、可用蓄热量、单周期空气需热、换热效率、空气预热温度、压损和建议换向周期。

### 7.10 多工况计算界面

`多工况计算` 用于比较设计工况、高产工况、低温装炉和厚坯合金钢工况。系统按产量、装炉温度、目标温度、在炉时间、炉温制度、效率和燃料热值计算各工况燃耗、单耗、节奏、升温裕度、表心温差和风险排序。

输出图示包含多工况趋势图和工况风险矩阵。

### 7.11 辊强度计算界面

#### 7.11.1 界面目的

`辊强度计算` 用于辊底炉炉辊强度和挠度校核。

#### 7.11.2 主要输入

1. 支承跨距。
2. 辊外径和辊内径。
3. 辊身有效长度。
4. 工件质量。
5. 同时承载辊数。
6. 动载系数。
7. 高温强度折减系数。
8. 材料常温屈服强度。
9. 弹性模量。
10. 允许挠度。
11. 最小安全系数。
12. 单根炉辊自重。
13. 轴承中心距。

#### 7.11.3 主要输出

1. 单辊等效载荷。
2. 最大弯矩。
3. 弯曲应力。
4. 剪应力。
5. 等效应力。
6. 高温许用应力。
7. 最大挠度。
8. 安全系数。
9. 辊强度图示。
10. 强度和挠度判定。

## 8. 图示输出说明

平台计算结果常见图示包括：

| 图示类型 | 用途 |
| --- | --- |
| 升温曲线 | 展示钢坯表面、中心或平均温度随时间变化 |

| 热平衡图 | 展示热收入、总热量和热支出构成 |
| 系统示意图 | 展示排烟路径、管束结构、管道流速或设备结构 |
| 载荷示意图 | 展示水梁、炉辊、步进框架等受力结构 |
| 风险矩阵 | 展示多工况燃料负荷和温差负荷风险 |

图示会随当前输入参数和计算结果自动刷新，并写入 PDF 计算报告。

## 9. 最近 5 次计算比较与最终计算

二级离线模型支持最近 5 次计算比较。

### 9.1 查看比较

1. 完成一次二级离线模型计算。
2. 点击结果卡片中的 `查看计算对比`。
3. 页面滚动到最近 5 次计算比较表。
4. 查看不同方案的炉温设定、出炉温度、表心温差和判定状态。

### 9.2 设置最终计算

1. 在比较表中选择目标计算结果。
2. 点击 `设为最终`。
3. 系统标记该结果为最终计算。
4. 后续报告归档时，以最终计算作为推荐方案。

最近 5 次计算记录保存在当前浏览器会话中，刷新页面后重新记录。

## 14. 项目资料操作

### 14.1 功能目的

`项目资料` 用于按工程项目归档资料，资料可作为 AI 问答和项目审查的依据。

### 14.2 资料类型

固定资料类型包括：

1. `现场反馈`
2. `审图单`
3. `技术说明`
4. `图纸目录`
5. `材料表`
6. `专利等技术文档`

### 14.3 单条上传

1. 点击左侧 `项目资料`。

7. 点击 `确认上传`。
8. 系统生成资料条目，并记录附件名称、类型和大小。

## 17. 常见操作问题

### 17.1 计算按钮置灰

原因：尚未选择工程项目。

处理：在 `计算管理` 顶部先选择工程项目。

### 17.2 报告窗口未打开

原因：浏览器拦截弹窗。

处理：允许当前站点打开弹窗后重新点击 `下载 PDF 计算报告`。

### 17.3 项目资料上传区置灰

原因：尚未选择工程项目。

处理：在 `项目资料` 顶部选择工程项目。

### 17.4 二级模型没有比较数据

原因：当前会话内计算次数不足或页面已刷新。

处理：重新执行多次计算后点击 `查看计算对比`。

### 17.5 计算结论提示调整

原因：输入参数导致温度、应力、挠度、压损、流速或安全系数未达到校核目标。

处理：复核输入参数，并按工程建议调整炉温制度、在炉时间、结构尺寸、材料等级、管径、支承跨距或安全系数。

## 18. 操作规范建议

1. 每次计算前确认工程项目选择正确。
2. 修改参数后重新计算，并查看图示是否同步更新。
3. 二级离线模型建议至少保留多个方案用于最近 5 次比较。
4. 最终归档前应设置最终计算。
5. PDF 报告下载后应按项目、炉型、模块和日期命名保存。
6. 现场反馈、审图单、技术说明、图纸目录、材料表和专利等技术文档应按资料类型分类上传。
7. AI 问答结果可作为资料查阅记录，与计算报告和审批意见一并归档。
