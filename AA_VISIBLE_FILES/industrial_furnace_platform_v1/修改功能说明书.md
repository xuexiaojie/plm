# 工业炉计算平台 V1 修改功能说明书

## 1. 文档目的

本文档说明本轮在保留主界面其它模块的基础上，补回全局登录入口，并将 `计算名目管理` 改名为 `计算管理`。原独立 `计算执行` 模块已移除，计算动作统一从 `计算管理` 进入。项目管理、横向对比、审批报告、数字孪生、项目资料、AI 联合分析、权限分配等其它模块保持原有页面和功能。

## 2. 主流程

当前主流程为：

```text
登录 → 主界面 → 计算管理 → 选择工程项目 → 固定炉型标签 → 固定计算按钮 → 对应程序占位界面
```

打开页面后先显示登录界面。点击 `登录进入主界面` 后，进入左侧导航主界面。

## 3. 调整范围

本轮调整范围限定为计算相关入口。

调整目标：

1. `计算名目管理` 改名为 `计算管理`。
2. 移除独立 `计算执行` 模块。
3. 在 `计算管理` 中先选择工程项目。
4. 固定三类炉型标签。
5. 固定每类炉型下的计算功能按钮。
6. 关闭计算名目自定义新增、删除、修改入口。
7. 点击计算按钮后进入对应程序占位界面。

## 4. 计算名目管理入口

页面入口：左侧导航 → `计算管理`。

该页面内部结构为：

```text
计算管理 → 选择工程项目 → 顶部固定炉型标签 → 固定计算按钮 → 对应程序占位界面
```

进入具体计算前必须先选择工程项目。程序占位界面会显示当前选择的工程项目、炉型和计算功能。未选择工程项目时，点击计算按钮会提示 `请先选择工程项目`。

## 5. 固定炉型标签

炉型标签固定为 3 个，顺序固定：

1. `步进炉`
2. `辊底炉`
3. `环形炉`

规则：

1. 炉型标签不可新增。
2. 炉型标签不可删除。
3. 炉型标签不可修改名称。
4. 炉型标签顺序固定。

后续接入真实计算程序时，建议按以下方式推进：

## 12. 步进炉二级离线模型

`步进炉二级离线模型` 按钮已接入独立计算模块。用户选择工程项目后，点击 `步进炉` 下的 `步进炉二级离线模型`，页面弹出新的计算界面，展示输入表单和计算结果区域。

当前模型为工程原型，参考加热炉二级控制和离线温度预报模型思路，采用以下计算结构：

1. 按预热段、加热段、均热段设置炉温。
2. 将钢坯厚度方向离散为一维节点。
3. 使用非稳态导热方程计算钢坯内部温度演化。
4. 两侧边界采用辐射换热和对流换热叠加。
5. 输出平均出炉温度、表面温度、心部温度、表心温差、钢坯质量、燃气量估算和步进节奏参考。
6. 使用 Canvas 绘制平均温度、表面温度、心部温度和目标温度计算曲线。
7. 使用 Canvas 绘制厚度方向温度随时间演化的温度云图。
8. 计算完成后提供 `下载 PDF 计算报告` 按钮，报告包含封面、工程名称、输入参数、结果摘要、计算曲线和温度云图，可通过浏览器打印另存为 PDF。

## 13. 步进炉加热曲线

`加热曲线` 按钮已接入弹出式计算模块。用户选择工程项目后，点击 `步进炉` 下的 `加热曲线`，页面弹出新的计算界面。

参数录入按三张参考图依次组织为：

1. `炉子参数`：炉子产量、钢坯入炉温度、结构尺寸、蓄热参数、燃料设定和炉温设定。
2. `料坯及空煤气`：钢坯规格、钢种成分、空气参数、煤气组成和煤气热值。
3. `散热损失`：炉墙散热、水冷散热、炉门孔洞散热和其他热损失。

当前计算逻辑采用分段炉温制度、一维非稳态导热、辐射与对流综合换热，并根据散热损失、水冷损失、孔洞损失和其他损失估算综合损失系数与煤气量。计算结果输出出钢表面温度、出钢平均温度、出钢中心温度、出钢平均温差、钢坯最大温差、最大温度应力、能耗、排烟温度、煤气量估算，并绘制加热曲线和温度云图。

参考计算结果截图的主要校核项为：出钢表面温度 `1253℃`、出钢平均温度 `1239℃`、出钢中心温度 `1224℃`、出钢平均温差 `29℃`、加热时间 `2.766h`、最大温度应力 `221MPa`、能耗 `329kcal/kg`、排烟温度 `888℃`。当前默认加热时间已校准为 `166min`，等效导热系数默认设为 `22 W/(m·K)`，用于贴近该参考结果。

热平衡图已按参考图片改为图示模式，包含左侧 `热收入`、中间 `总热量`、右侧 `热支出`。热收入项包括 `燃料物理热`、`空气物理热`、`钢坯氧化放热`、`燃料化学热`；热支出项包括 `钢坯吸热`、`炉气带走热量`、`炉衬散热`、`水冷散热`、`炉门及孔洞散热`、`其他散热损失`。默认百分比按参考分析图录入，小时热量根据当前钢坯吸热和产量动态换算。

计算完成后提供 `下载 PDF 计算报告` 按钮，报告包含封面、工程名称、输入参数、计算结果摘要、加热曲线、温度云图和热平衡图，可通过浏览器打印另存为 PDF。

## 17. 步进炉水梁计算
