# 12 权限矩阵规格

## 1. 角色

| 角色 | 说明 |
|---|---|
| 普通计算人员 | 项目建模和计算执行 |
| 审核人 | 审核结果和报告 |
| 模板管理员 | 管理模板和字段 |
| 算法维护人员 | 管理算法入口和样例 |
| 系统管理员 | 管理用户、角色、字典 |
| 只读用户 | 查看授权数据 |

## 3. 页面按钮

1. 无权限按钮隐藏。
2. API 必须二次校验权限。
3. 权限不足返回 `PERMISSION_DENIED`。
